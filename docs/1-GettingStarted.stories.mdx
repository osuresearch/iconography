# Office of Research Iconography

This is a collection of icons from https://iconify.design/ that are common across Office of Research applications.

> For additional icons to be added to the collection, open a feature request.

All icons are SVG and can be rotated, scaled, and recolored.

## Installation

```
npm i @osuresearch/iconography
```

## Usage

```tsx
import { Icon } from '@osuresearch/iconography';
...

render() {
  return (
    <Icon />
  )
}
```
